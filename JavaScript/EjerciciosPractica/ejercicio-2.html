EJERCICIOS PRACTICA # 3 JAVASCRIPT

1. Crear una funcion que reciba como parametro una cadena de texto e imprima la cadena de texto en mayusculas. Ejemplo: "Bienvenido" -> "BIENVENIDO".

2. Crear una funcion que reciba como parametro una cadena de texto e imprima la cadena de texto en minusculas. Ejemplo: "BienVenIdO" -> "bienvenido".

3. Crear una funcion que reciba un texto e imprima la longitud de la cadena de texto. Si la longitud es mayor que 5, que se imprima la cadena recortada. Ejemplo: "Angel Arevalo" -> "Angel"

4. Crear una funcion que reciba un texto e imprima la primera letra y la ultima. Ejemplo: "Cadena de texto" -> "C <-> o"

5. Crear una funcion que reciba un texto e imprima la palabra "durazno" si se incluye en el texto Ejemplo: "Quiero comer durazno" -> "durazno" รณ "no hay durazno" (si no esta la palabra durazno)
